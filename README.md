# GrievanceGrid

GrievanceGrid is a smart public service CRM that centralizes citizen complaints, automates routing and task assignment, and tracks resolution in real time. It improves transparency, accountability, and efficiency in public grievance management.

---

## Features

- Centralized complaint submission  
- Intelligent auto-routing to departments  
- Real-time status tracking  
- SLA monitoring and escalation  
- Geo-tagging and heatmaps  
- Citizen, Officer, and Admin dashboards  
- Firebase Authentication (Phone OTP + Google)  
- Scalable, cloud-ready architecture  

---
